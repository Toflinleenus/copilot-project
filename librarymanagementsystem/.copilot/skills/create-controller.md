# Skill: Create Controller

Create a Spring Boot REST controller with:

- @RestController
- @RequestMapping with plural resource name
- CRUD endpoints (POST, GET, PUT, DELETE)
- Use ResponseEntity
- Use @Valid and BindingResult
- Call Service layer
- Proper status codes

Example:
POST → 201 CREATED  
GET → 200 OK  
PUT → 200 OK  
DELETE → 204 NO CONTENT
